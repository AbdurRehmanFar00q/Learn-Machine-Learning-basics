# 📊 Student Score Predictor

A complete **Linear Regression** model that predicts student exam scores based on study hours. Built with **Scikit-learn**, featuring user input, model evaluation, and data visualization.

---

## 🎯 Project Overview

This project demonstrates a complete Machine Learning workflow:
- ✅ Load and explore data
- ✅ Train a Linear Regression model
- ✅ Evaluate model performance (MSE, R²)
- ✅ Take user input for real-time predictions
- ✅ Visualize results with matplotlib

---

## 📈 Model Performance

| Metric | Score |
|--------|-------|
| **R² Score** | 0.9966 (99.66%) |
| **Mean Squared Error (MSE)** | 0.9509 |
| **Model Type** | Linear Regression |
| **Train-Test Split** | 80% / 20% |

### 🔍 Interpretation
- **R² = 0.9966** → Model explains 99.66% of the variance
- **MSE = 0.95** → Very low prediction error
- **Conclusion:** Excellent model performance! 🌟

## 📂 Dataset

The dataset contains **study hours vs exam scores**:
- **Feature (X):** Hours studied
- **Target (y):** Exam score
