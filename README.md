# 📘 Learn Machine Learning Basics

A beginner-friendly repository to learn **Machine Learning fundamentals**, explore **essential libraries**, practice **different ML methods**, and build **hands-on projects**.

## 🔹 Overview

This repo contains three main sections:

1. **Libraries** – Basics of NumPy, Pandas, Matplotlib, and Scikit-learn  
2. **ML Methods** – Regression, Classification, Clustering  
3. **Projects** – Hands-on ML projects organized by method  

## 📂 Structure

- `libraries/` – Mini-projects for each ML library  
- `machine-learning-projects/` – Projects for regression, classification, clustering  
- `requirements.txt` – Install all dependencies

## 🛠️ Libraries Covered

| Library | Description |
|---------|-------------|
| **NumPy** | Numerical computing, arrays, matrix operations |
| **Pandas** | Data manipulation, DataFrames, CSV handling |
| **Matplotlib** | Data visualization, line plots, scatter plots |
| **Scikit-learn** | Machine Learning algorithms, model evaluation |

## 📊 ML Methods & Projects

- **Regression:** Linear Regression 
  - Student Score Predictor  
  - House Price Prediction  

- **Classification:** Logistic Regression, KNN, Decision Tree  
  - Spam Classifier  
  - Iris Flower Classifier  

- **Clustering:** K-Means  
  - Customer Segmentation
